# Encrypted Chat

## Project Statement:

Create a GUI chat with a user nicknames list, send encrypted personal messages not readable on the server, and encrypted broadcast messages.

## Libraries:

- Tkinter for testing then CustomTkinter
- Socket
- Threading
- RSA
