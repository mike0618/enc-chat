# Encrypted Chat

## Project Statement:

Create a GUI chat with a user nicknames list, send encrypted messages directly to users and unencrypted broadcast messages.

## Libraries:

- Tkinter for testing then CustomTkinter
- Socket
- Threading
- RSA
